
## 解析
打到目的地的每一跳都打印出来
- 序号
- 第一个包往返时间
- 第二个包往返时间
- 第三个包往返时间
- 这一跳的地址
为什么traceroute比ping慢，只有第10跳的时间才是和ping的时间相同
