## 1. oh-my-zsh安装
```sh
yay -S oh-my-zsh
```
## 2. 插件配置
vim ~/.zsh
```
source ~/.bashrc
plugins=(git zsh-syntax-highlighting zsh-autosuggestions sudo extract fzf cp colored-man-pages zsh_reload archlinux docker golang pip npm z)
```
## 3. 生效
```
source ~/.zshrc
```
