## 1. Linux的内存管理
## 2. 交换空间
是磁盘的一个分区，Linux内存满时，会把一些内存交换只Swap空间。
### 2.1. 交换空间 vs 虚拟内存

|     |    交换空间    |    虚拟内存    |
| --- | ------------- | ------------- |
|     | 存在于磁盘中   | 存在于磁盘中   |
|     | 与主存发生置换 | 与主存发生置换 |
|     | 操作系统概念   | 进程概念       |
|     |       解决系统物理内存不足的问题        |        解决进程物理内存不足的问题       |
