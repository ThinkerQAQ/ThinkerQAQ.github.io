## 2. 排序对比
|     | 时间复杂度 | 空间复杂度 | 是否原地排序 | 稳定性 |
| :-- | :-------- | :-------- | :---------- | :----- |
| 冒泡 | O(N²)     | O(1)      | √           | √      |
| 选择 | O(N²)     | O(1)      | √           | ×      |
| 插入 | O(N²)     | O(1)      | √           | √      |
| 归并 | O(NlogN)  | O(N)      | ×           | √      |
| 快速 | O(NlogN)  | O(logN)   | √            | ×      |
| 堆   | O(NlogN)  | O(1)      | √            | ×      |
